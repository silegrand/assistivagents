How it works

What is a FEP score?

You don't need to be a data scientist to use these tools. Here's what's happening under the hood — in plain English.

Think of it like a credit score — but for frailty risk in a community. We look at 17 different things about an area. Each one gets a number between 0 and 100, where 50 means exactly average for England. Higher than 50 means that area has more risk than England average. We then blend all 17 together — some counted more heavily than others — to get one final score.

1

We measure 17 things

Falls, hip fractures, winter deaths, loneliness, dementia diagnosis rate, antidepressant prescribing and more — all from real NHS sources.

2

We put them on one scale

Each signal is converted to a 0–100 number. 50 = England average. 60 = 20% worse than England. This lets us compare falls rates and prescribing side by side.

3

We weight by importance

Living alone at 75+ carries 13% of the final score — the strongest single predictor. Diuretic prescribing carries 1%. All 17 weights add up to exactly 100%.

4

One score per district

The weighted average of all 17 signals becomes the FEP score. A score of 65 means that district is substantially more at-risk than England across multiple measures simultaneously.

System Architecture

Five layers. One proactive pathway.

This tool suite is Layer 2 of a five-layer population frailty intelligence system. Each layer feeds the next — from public data ingestion through to clinical triage and feedback.

L1
Population Signal Ingestion
ONS Census 2021, NHS Fingertips (10 indicators confirmed), IMD 2019, CQC register, NHSBSA English Prescribing Dataset (7 signal groups confirmed — including anticholinergic burden). 17 signals total — 14 real, 3 synthetic.
Active — 14 real signals confirmed Active — data confirmed
L2
Composite FEP Scoring — this tool
Weighted composite index (17 signals, 14 real) generating a Frailty Emergence Probability score per district. Geographic heat map. Commissioner briefing export. ICB baseline grounding via NHS Fingertips + NHSBSA EPD.
Active — prototype deployed Active — prototype deployed
L3
Community Touchpoint Identification
Routes outreach to GP surgeries, pharmacies, libraries, faith communities, and VCSE organisations within priority zones. Google Places API integration.
Planned — Phase 2
L4
Precision Screening — resiliencetools.xyz
Voice-first PRISMA-7 screening, carer observational report, Triple Tap Intelligence synthesis, Frailty Profile Radar. Context-injected with zone FEP score to calibrate sensitivity thresholds.
Active — live at resiliencetools.xyz
L5
Triage, Referral & Feedback Loop
Clinical referral, population dashboard, Assistiv onboarding pathway, safety escalation. Anonymised screening outcomes recalibrate the FEP model — the dataset becomes the competitive advantage.
Planned — Phase 3
Data Sources

Public data. Defensible intelligence.

Every signal in the FEP model uses publicly available data — free to access, citable in a commissioner briefing, and updatable without licensing costs. The value is in the composite, not the inputs.
