System Architecture

Five layers. One proactive pathway.

This tool suite is Layer 2 of a five-layer population frailty intelligence system. Each layer feeds the next — from public data ingestion through to clinical triage and feedback.

L1
Population Signal Ingestion
ONS Census 2021, NHS Fingertips, IMD 2019, DWP benefits data, CQC register, GP dispensing data. Eight public sources normalised to a common spatial index.
Active — data confirmed
L2
Composite FEP Scoring — this tool
Weighted composite index generating a Frailty Emergence Probability score per district. Geographic heat map. Commissioner briefing export. ICB baseline grounding.
Active — prototype deployed
L3
Community Touchpoint Identification
Routes outreach to GP surgeries, pharmacies, libraries, faith communities, and VCSE organisations within priority zones. Google Places API integration.
Planned — Phase 2
L4
Precision Screening — resiliencetools.xyz
Voice-first PRISMA-7 screening, carer observational report, Triple Tap Intelligence synthesis, Frailty Profile Radar. Context-injected with zone FEP score to calibrate sensitivity thresholds.
Active — live at resiliencetools.xyz
L5
Triage, Referral & Feedback Loop
Clinical referral, population dashboard, Assistiv onboarding pathway, safety escalation. Anonymised screening outcomes recalibrate the FEP model — the dataset becomes the competitive advantage.
Planned — Phase 3
Data Sources

Public data. Defensible intelligence.

Every signal in the FEP model uses publicly available data — free to access, citable in a commissioner briefing, and updatable without licensing costs. The value is in the composite, not the inputs.

ONS · Free
Census 2021
Over-75s living alone at LSOA level. Strongest single demographic frailty proxy. ~1,500 households per zone.
Live — confirmed
OHID · Free API
NHS Fingertips
Falls admissions, unplanned hospitalisations, healthy life expectancy. ICB-level data confirmed. Sub-ICB linkage in Phase 2.
ICB level — confirmed
MHCLG · Free
IMD 2019
Index of Multiple Deprivation. Access and vulnerability amplifier. Available at ward and LSOA level across all Kent districts.
Live — confirmed
DWP · Free
Attendance Allowance
DWP benefit claims by postcode district. Functional limitation indicator. Published quarterly, geographic granularity confirmed.
Live — confirmed
CQC · Free API
Care Home Register
Care home locations, capacity, and CQC ratings. Service gap indicator — high FEP plus low care provision identifies priority zones.
Live — confirmed
NHS · Free
GP Dispensing Data
Polypharmacy rates (5+ medications) by practice. Strong clinical frailty marker. Published by NHS Business Services Authority.
Practice level — confirmed
